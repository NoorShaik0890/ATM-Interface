# ATM Interface Program

## Overview
This Java program simulates the functionalities of an ATM machine, providing users with a secure and intuitive way to perform banking operations.

## Instructions
To run the program:
1. Compile the Java files using a Java compiler (e.g., `javac Main.java`).
2. Run the compiled program using the Java interpreter (e.g., `java Main`).
3. Follow the prompts to enter your customer number, PIN number, and select your account type to perform banking operations.

## Features
- User authentication based on customer numbers and PINs
- Checking and savings account management
- Operations include checking balance, withdrawing money, and depositing money
- Error handling mechanisms for invalid input and insufficient funds

## Usage
- Upon running the program, users will be prompted to enter their customer number and PIN number for authentication.
- After successful authentication, users can select their account type (checking or savings) to perform operations.
- Follow the on-screen instructions to perform desired banking operations.

## Notes
- For demonstration purposes, valid customer numbers and PINs are hardcoded in the `User` class.
- The program does not persist data between runs and is for educational purposes only.
